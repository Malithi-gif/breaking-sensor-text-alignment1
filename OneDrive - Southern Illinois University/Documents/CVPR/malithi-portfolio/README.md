# Malithi Mithsara Personal Portfolio Website

This is a responsive personal academic portfolio website created from the provided CV and photo.

## Files

- `index.html` — website structure
- `style.css` — visual design and responsive layout
- `script.js` — interactive features
- `assets/profile.jpg` — profile photo
- `assets/Malithi_Mithsara_CV.pdf` — downloadable CV

## How to run locally

Open `index.html` in a browser.

For a local server, run:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## How to deploy with GitHub Pages

1. Create a GitHub repository.
2. Upload all files in this folder to the repository root.
3. Go to repository **Settings** → **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch **main** and folder **/root**.
6. Save.
7. Your website will be available at:

```text
https://YOUR-GITHUB-USERNAME.github.io/YOUR-REPOSITORY-NAME/
```
